# FunHub 

A fun full-stack web application built to practice and improve my frontend and backend development skills using React, Vite, TailwindCSS, and Spring Boot.

FunHub is a personal playground project where I experiment with API integrations, UI design, responsive layouts, and backend development while building entertaining features like joke generators, quote generators, meme generators, and more.

---

# Purpose of the Project

This project is mainly focused on helping me improve my skills in:

## Frontend
- React
- Vite
- TailwindCSS
- Responsive Design
- Component-based architecture
- State management
- API integration using Axios
- UI/UX design

## Backend
- Spring Boot
- REST APIs
- External API consumption
- JSON handling
- Backend architecture
- Error handling

# Getting Started

## Clone the Repository

```bash
git clone https://github.com/AidenClintonWallace/The-FunHub-FrontEnd.git
```

---

# Frontend Setup

```open in VsCode
npm install
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```

---

# Backend Setup(must clone backend repo)

Run the Spring Boot application from your IDE :

Backend runs on:

```text

```
